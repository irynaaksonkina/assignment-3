Створені таблиці: customers, products, orders, order_items, order_log.
Функція calculate_order_total рахує суму замовлення.
Процедура create_order створює нове замовлення.
Процедура add_product_to_order додає товар до замовлення.
Тригер на order_items автоматично оновлює суму замовлення.
Тригер на orders записує лог при створенні замовлення.
Також додані тести, які перевіряють роботу всіх процедур, функцій та тригерів.
